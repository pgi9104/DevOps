<h1>앤서블 모듈</h1>

## 1. 공통 모듈
- name: 명명
- hosts: 호스트

## 2. 파일을 관리하는 모듈
- path: 관리되어야할 객체가 존재하는 절대 경로를 작성
- state: path에 명시된 경로가 파일인지 디렉토리인지 확인
  - directory: 디렉토리 생성
  - file: 파일이 존재하는 지 확인.
  - touch: file을 생성시 사용.
  - link: 소프트 링크를 생성한다.
  - hard: 하드 링크를 생성한다.
  - absent: path에 기록되어 있는 파일/디렉토리를 삭제.
- mode: 생성할 디렉토리/파일의 권한을 설정하며 8진수이며 제일 앞에 0을 붙여줘야 한다.
- owner: 소유자를 지정한다.
- group: 소유그룹을 지정한다.
